## 🖥️ 'The System Blueprint' 최종 목업 디자인 사양

Writer가 확정한 핵심 메시지("AI 시대, 당신은 데이터에 휘둘리는 운영자가 아니라, 성장을 통제하는 시스템 설계자입니다.")와 Silver Tier의 가치를 극대화하기 위해, **'The Control Panel'** 컨셉을 확장하여 시스템 통합 거버넌스(System Integration Governance)를 시각적으로 구현하는 'The System Blueprint' 목업 디자인 사양을 제시합니다.

이 사양은 Figma에 직접 적용 가능한 구체적인 레이아웃과 비주얼 요소를 포함합니다.

---

### 1. 디자인 컨셉 및 분위기 (Aesthetics & Tone)

*   **컨셉:** 시스템 제어판(Control Panel) + 청사진(Blueprint). 복잡한 데이터 흐름을 명확하고 통제 가능한 구조로 시각화하여, 고객에게 '통제권 확보'라는 심리적 안정감을 전달합니다.
*   **톤앤매너:** 전문적(Professional), 신뢰감(Trustworthy), 정밀함(Precision), 미래지향적(Futuristic).
*   **주요 색상 팔레트 (Color Palette):**
    *   **Primary (시스템 기반):** Deep Navy Blue (`#0A1931`) - 깊이와 신뢰감을 표현.
    *   **Secondary (액션/경고):** Electric Cyan (`#00FFFF`) - 데이터 흐름, 통제권(Control)의 활성화를 상징.
    *   **Accent (안정성):** Light Slate Gray (`#E0E0E0`) - 배경 및 정보 영역으로 사용.
    *   **Highlight (성장/핵심):** Bright Gold/Amber (`#FFC72C`) - 핵심 지표, 통제권을 확보했다는 성과를 강조.

### 2. 타이포그래피 시스템 (Typography System)

*   **헤드라인 (H1/H2):** **Montserrat Bold** 또는 **Inter Black**. 크고 강렬하게 '통제권'의 메시지를 전달합니다.
*   **본문 및 데이터 (Body/Data):** **Roboto Mono** 또는 **Fira Code**. 기술적이고 정밀한 느낌을 주어 거버넌스 및 시스템 측면의 신뢰도를 높입니다.

### 3. 레이아웃 구조 상세 (Layout Structure: The System Blueprint)

**전체 레이아웃:** 16:9 또는 16:10 와이드 포맷을 기반으로 하며, 좌측에 통제 영역(Control Panel UI), 우측에 시스템 흐름 시각화(Blueprint)를 배치합니다.

#### A. 좌측 패널: 컨트롤 패널 (The Control Panel - 인터페이스 영역)

*   **배경:** Deep Navy Blue (`#0A1931`).
*   **상단 바:** 로고 및 현재 모듈 명시.
    *   `[SOO-LYEAN'S AI | SYSTEM BLUEPRINT]` (Cyan 또는 Gold 텍스트)
*   **핵심 제어 영역 (Control Area):**
    *   **메인 타이틀:** "AI 거버넌스 대시보드" (H1, White).
    *   **통제 지표 카드 (KPI Cards - 3개):** 각 카드는 Gold 테두리를 가지며 아래와 같이 구성됩니다.
        1.  **리스크 점수:** `[Risk Score]` (숫자: 75/100) — Cyan 색상으로 위험 수준 표시.
        2.  **통제 범위:** `[Control Scope]` (텍스트: Full / Partial / Fragmented) — 현재 상태 시각화.
        3.  **거버넌스 통합도:** `[Governance Integration]` (시각적 게이지 바, Gold 색상으로 채워짐).
    *   **액션 버튼:** 중앙 하단에 가장 강조되는 CTA 배치.
        *   `[Execute Governance Roadmap] (Primary Button - Electric Cyan 배경)`

#### B. 우측 패널: 시스템 청사진 (The Blueprint - 흐름 시각화 영역)

*   **배경:** Light Slate Gray (`#E0E0E0`) 또는 매우 옅은 회색 그라데이션.
*   **핵심 비주얼:** 데이터 및 모델의 **흐름(Flow)**을 표현하는 추상적인 네트워크 다이어그램/청사진 스타일.
    *   **중앙 요소:** 데이터 흐름이 시작되는 지점(Input)에서 최종 결과물(Output)까지의 경로를 선(Line)과 노드(Node)로 표현합니다.
    *   **노드 예시:** 'Raw Data' → 'AI Model Training' → 'Audit Log Verification' → 'Business Outcome'. 각 단계는 시스템 통합이 이루어진 지점임을 점선으로 표시합니다.
    *   **통합 강조:** 데이터 흐름 중간에 **Cyan 선**을 사용하여, 거버넌스 레이어(Governance Layer)가 모든 프로세스를 감싸고 있음을 시각적으로 강조합니다.

### 4. 최종 디자인 요약 (Mockup Summary)

| 요소 | 디자인 적용 사항 | 의도 |
| :--- | :--- | :--- |
| **전체 분위기** | 어두운 배경(Navy) + 네온 블루/골드 포인트 | 신뢰감과 첨단 기술의 결합 |
| **KPI 카드** | Gold 테두리, Cyan 지표 강조 | 성과와 통제권 확보를 즉각적으로 인지 |
| **흐름 시각화** | 청사진 스타일의 노드 연결 다이어그램 | 복잡한 시스템을 단순하고 명확하게 이해 |
| **CTA 버튼** | Electric Cyan 배경 (가장 밝은 색) | 사용자가 다음 행동(시스템 시작)으로 즉시 유도 |
